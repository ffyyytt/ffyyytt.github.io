Misc/reCAPTCHA v39

I was browsing the web and happened to see a website that uses reCAPTCHA v39. I tried to solve it, but failed. Can you help me out? Surely it's much harder than v3...
http://172.86.96.174:8000

             import cv2
import time
import numpy as np
import matplotlib.pyplot as plt
from selenium import webdriver
from selenium.webdriver.common.by import By
driver = webdriver.Chrome()
driver.get("http://172.86.96.174:8000")
time.sleep(3)
button = driver.find_element(By.ID, "start")
button.click()
while True:
    img = driver.find_element(By.ID, "display")
    time.sleep(2)
    with open("captcha.png", "wb") as f:
        f.write(img.screenshot_as_png)
    image = cv2.imread("captcha.png")
    c = np.mean(image == [240, 240, 240])*100
                
    answer = driver.find_element(By.ID, "answer")
    answer.send_keys(99-c)
    
    submit = driver.find_element(By.ID, "submit")
    submit.click()
